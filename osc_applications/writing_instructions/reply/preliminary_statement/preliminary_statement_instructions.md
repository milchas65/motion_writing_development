# Preliminary Statement Instructions (OSC Application - Reply)

## Purpose
Rebut opposition's arguments and reinforce entitlement to injunctive relief.

## Structure

Use the "confirms" framing:

```
PRELIMINARY STATEMENT

[Opposition's] opposition confirms [admissions favorable to movant]. [Opposition's] arguments fail because [preview rebuttals].
```

## Content Elements

### Highlight Admissions
- Facts opponent did not dispute
- Concessions supporting injunction elements

### Rebut Defenses
- Address claimed lack of success
- Rebut irreparable harm arguments
- Counter balance of equities claims
